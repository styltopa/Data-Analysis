Άσκηση 4η
- Βρίσκουμε το σταστικό στο αρχικό δείγμα διμεταβλητών παρατηρήσεων και κάνουμε bootstrap ή κάνουμε boostrap στα δείγματα ξεχωριστά και μετά παίρνουμε το bootstrap για καθένα από αυτά;

Για την περίπτωση που θέλεις να κάνεις bootstrap για τον συντελεστή συσχέτισης, πρέπει τα δείγματα να μην είναι ανακατεμένα γιατί αλλιώς χάνεται ούτως η άλλως η όποια συσχέτιση των δεδομένων.

Έτσι, πρέπει πρώτα να υπολογίσεις το στατιστικό στο διμεταβλητό δείγμα και μετά να πάρεις bootstrap του αρχικού στατιστικού.

Όταν έχεις μόνο ένα δείγμα, ο μέσος όρος του ΜΣ Fisher είναι η μία τιμή του στατιστικού που υπολογίστηκε από το δείγμα.


- Όταν λέει έλεγχο τυχαιοποίησης για μηδενικό συντελεστή συσχέτισης εννοεί το εξής:
Αυτός ο έλεγχος μπορεί να γίνει μόνο για έλεγχο μηδενικής συσχέτισης. 
Αυτό που κάνουμε είναι ότι επιλέγουμε τη μία μεταβλητή και την αντιμεταθέτουμε. 
Έτσι καταστρέφεται η όποια συσχέτιση υπήρχε προηγουμένως.

Υπολογισμός p-value από δείγματα bootstrap ή τυχαιοποίησης:
Αν rStat <= median(randomisedCorrCoefs)
	pVal = 2*(r/(B+1)) 
else
	pVal = 2(1-r/(B+1))
